Hi, My name is Yu Yang. I am a full stack web developer and geologist who loves coding and data analysis and visualization.
Oil Trend is Web Application dynamically showing annual oil trends of consumption, production, import, export across all world from 1980 to 2014. The data source is from EIA. The app is built by using HTML, CSS, JS, jQuery, AJAX, BootStrap, Node.js, MongoDB, etc.
Daytrippr, a team project and social media application, enables users to make trip plans together with friends and potential participants. It was developed within a week using Ruby, Rails, JavaScript, PostgreSQL and BootStrap frameworks. I led and took charged of key features include user authentication, Google Map markers, emailing, commenting and posting, and generating trip routes.
