Portfolio

MindStaq

MindStaq is a complicated online software system which I am currently working on as my first full-time software developer job. The front-end framework of most apps of the system is Angularjs and I am taking charge of building Calendar, Task, Search UI components and related services for the main app by using Fullcalendarjs library and Nylas api. In the mean time, I am also participating in designing and implementing codes on the back-end which use Nodejs Express as the framework. These works enable me to get picture of Microservice architecture and begin to think about common challengings of software developing like Scalability Concurrency Callback hell. For example, the real-time search service I am responsible to implement is through RabbitMQ for events publish / subscribe, asynchronous processing, or messaging queues between MongoDB operation in the main Microservice and ElasticSearch operation in the search Microservice. I also use nodejs async to handle Callback hell issues during implementing ElasticSearch.

Oil Trends

This personal project enabled me further understand Thick client architecture. Building rich features and functions of single page application by just using jQuery and ajax was challenging but rewarding. Another challenging thing was data preparation and formatting. Because in order to get oil trend, large datasets were inevitable. Yes, processing more than 30 years datasets of all kinds of oil data from EIA was painful, but I still successfully processed and loaded into MongoDB and served data api by using Nodejs Express Mongoose. The rest of works pretty much focused on the front-end. Thanks to D3, a powerful data visualization library, really makes data speek through all kinds of graphs, like showing annual oil trends of consumption, production, import, export across all world from 1980 to 2014. This app was deployed and hosted on DigitalOcean
